# CreditSea LMS - Frontend

This is the frontend portion of the CreditSea Loan Management System, built with Next.js 15, TypeScript, and Tailwind CSS.

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Environment Variables**:
   Create a `.env.local` file:
   ```env
   NEXT_PUBLIC_API_URL=http://localhost:5000/api
   ```

3. **Run Development Server**:
   ```bash
   npm run dev
   ```

For full project documentation, setup instructions, and workflow details, please refer to the [Main Project README](../README.md).
